BuckSites Tools — CSV column reference for scripts/import-seed.ts
===================================================================

Place files under ./data/ (or use --sample to read *.sample.csv).

1) brands.csv
   Required:
     slug   — URL-safe id (matches /brand/[slug])
     name   — Display name

2) filters.csv
   Required:
     brand_slug                  — Must exist in brands after import
     slug                        — URL-safe id (matches /filter/[slug])
     oem_part_number             — OEM part number string
   Optional:
     name                        — Marketing / short description
     replacement_interval_months — Integer months (empty = null)
     notes                       — Free text (empty = null)

3) fridge_models.csv
   Required:
     brand_slug                  — Must exist in brands
     slug                        — URL-safe id (matches /fridge/[slug])
     model_number                — As printed on unit / manual
   Optional:
     replacement_interval_months — Integer (empty = null)
     notes                       — Free text (empty = null)

4) compatibility_mappings.csv
   Required:
     fridge_slug  — fridge_models.slug
     filter_slug  — filters.slug
   (Composite key; re-importing the same pair updates nothing new.)

5) retailer_links.csv
   Required:
     filter_slug    — filters.slug
     affiliate_url  — Full https (or http) outbound URL
   Optional:
     retailer_name  — Label for the button (empty = null)
     is_primary     — true/false/1/0 (empty treated as false)
     sort_order     — Integer, default 0

Upsert behavior:
   retailer_links: same filter_slug + affiliate_url updates name/flags/order;
                   new URL rows are inserted.
